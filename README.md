# Movie-Genre-Classification
Overview:
This project predicts the genre of a movie based on its plot summary using a Machine Learning.

Technologies Used:
1.Python.
2.Pandas.
3.scikit-learn.
4.TF-IDF Vectorization.
5.Logistic Regression.

Dataset:
The project uses a movie dataset containing movie descriptions and genres.

Model Workflow:
1.Load Dataset.
2.Extract movies descriptions and genres.
3.Convert Text into TF-IDF features.
4.Train a Logistic Regression Model.
5.Predict movie genres from plot summaries.

Accuracy:
Model Accuracy:66.7%

How to Run:
Install require libraries:
       pip install -r requirements.txt
Run the Project:
       python main.py
